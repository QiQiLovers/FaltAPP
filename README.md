FaltAPP es un proyecto estudiantil creado para registrar licencias en un ambiente educativo, creado por Tomas Garula y Santiago Rosas.
Para usar el codigo necesitas usar un IDE de Java, preferiblemente Apache NetBeans 27+ con el conector mysql que esta en el proyecto.
Tambien sera necesario usar XAMPP para conectar la base de datos con el programa, utilizando el codigo de SQL que esta como 'faltappDB.sql' para crearla
